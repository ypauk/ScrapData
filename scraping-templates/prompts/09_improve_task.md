# Задача

Проведи ревью по следующим направлениям.

## 1. Качество кода scraper.py и parser.py

- Соответствует ли код принципам из AI_rules.md (простота, минимум зависимостей, requests/BS4 вместо Playwright, если возможно)?
- Есть ли избыточная сложность, дублирование логики с ядром (browser.py, config.py, utils.py, exporter.py)?
- Обработка ошибок — достаточна ли она? Не "проглатывает" ли она реальные проблемы?
- Соответствует ли код project_plan.md (имена функций, поля результата)?

## 2. Устойчивость и риски

- Учтены ли типичные риски: rate limit, антибот, пагинация, пустые страницы?
